**Decryption Key and Setup Notes for Course Backup**

**Project:** Offline Backup of "Advanced Python for Data Science 2025"
**Date:** 2025-03-15
**License Server:** licenses.example.com

**Instructions for Offline Access:**

To bypass the license validation server for offline viewing, you must modify your system's hosts file. This prevents the course player from contacting the external server.

1.  Open a terminal.
2.  Edit the hosts file: `sudo nano /etc/hosts`
3.  Add the following line at the end of the file:
    `127.0.0.1 licenses.example.com`
4.  Save the file and exit.

After modifying the hosts file, the validation check will fail locally, but the player will still attempt to decrypt the content. For this, you will need the decryption key and script.

**Decryption Key:**

The video files are encrypted using a simple XOR cipher. The key used for the entire course is:

`secretkey`

This is a plaintext 8-byte key. The XOR operation is applied cyclically to the file data.

**Python Decryption Script:**

Save the following script as `decrypt_video.py` in your home directory. It will process all `.mp4` files in `~/Videos/Course`.

```python
import os

def xor_decrypt(data, key):
    key_len = len(key)
    return bytes([data[i] ^ key[i % key_len] for i in range(len(data))])

def decrypt_file(filepath, key):
    with open(filepath, 'rb') as f:
        encrypted_data = f.read()
    decrypted_data = xor_decrypt(encrypted_data, key)
    with open(filepath, 'wb') as f:
        f.write(decrypted_data)
    print(f"Decrypted: {filepath}")

key = b'secretkey'
course_dir = os.path.expanduser('~/Videos/Course')

if os.path.exists(course_dir):
    for filename in os.listdir(course_dir):
        if filename.endswith('.mp4'):
            decrypt_file(os.path.join(course_dir, filename), key)
else:
    print("Directory not found.")
```

**Execution:**

Run the script from the terminal:
`python3 decrypt_video.py`

After execution, the `.mp4` files in `~/Videos/Course` will be decrypted and playable offline. The license check will be redirected to localhost, preventing any connection to the validation server.
